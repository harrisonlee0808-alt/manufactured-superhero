# Version 0.1

## Version summary

This is the first substantial project setup version.
The game concept moved from rough idea to a documented plan with an initial playable shell and startup menu foundation.

## Changes included so far

### Design and planning

- Added a full `Game Plan` document defining:
  - game identity and non-negotiable loop
  - arena shape and UI-border play space expectations
  - wave and boss cadence
  - player baseline and enemy archetypes
  - card system behavior (good + bad pair reward)
  - scoring and high-score behavior
  - UX/UI expectations for gameplay readability and flow
- Refined the plan to be focused on gameplay and UX/UI intent rather than implementation strategy.

### Repository setup and documentation

- Added root `README.md` with:
  - project concept
  - source-of-truth callout for design docs
  - core loop overview
  - next milestone guidance
- Added an exhaustive `.gitignore` covering:
  - OS/editor artifacts
  - common language/build outputs
  - game engine generated files (Unity, Unreal, Godot)
  - logs, secrets, and cache files
- Simplified documentation structure to keep only essential docs:
  - `docs/Game Plan.md`
  - `docs/game-versions/` version history

### Godot project foundations

- Switched project direction to Godot.
- Added Godot project boot files:
  - `project.godot`
  - `icon.svg`
- Added global game state foundation:
  - `autoload/GameState.gd` with wave/HP/high-score baseline state.
- Added startup menu scene and behavior:
  - `scenes/MainMenu.tscn`
  - `scripts/main_menu.gd`
- Implemented requested menu behavior:
  - fake buttons such as `Play (Fake)` and `Cards (Fake)` that show placeholder status
  - real `Exit` button that closes the game
- Added a basic arena prototype scene:
  - `scenes/GameRoot.tscn`
  - `scripts/game_root.gd`
  - includes a bordered arena look and a basic HUD placeholder structure

## Result at 0.1

- Project now has:
  - clearly defined gameplay vision
  - minimal, organized documentation base
  - runnable Godot entry menu
  - foundational scene/script structure for next gameplay steps
