# Versioning Rules

Game version format is:

- `major.minor` (example: `0.1`, `1.0`, `2.3`)

## Increment rules

- Increase **major** (`1.0 -> 2.0`) for big, game-defining changes.
  - Examples: core loop changes, major progression model changes, major perspective/control changes.
- Increase **minor** (`0.1 -> 0.2`) for smaller but meaningful changes.
  - Examples: new cards, wave tuning, UI flow updates, menu/UX additions, quality improvements.

## Current version
